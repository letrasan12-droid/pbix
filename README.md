# 📊 Desafio de Transformação de Dados com Power BI

Este repositório contém a solução para um desafio de **Transformação, Limpeza, Modelagem e Análise de Dados** com **Power BI**, utilizando **Power Query** e **DAX**. O objetivo é demonstrar boas práticas de ETL, modelagem dimensional (modelo estrela) e visualização de dados com indicadores e KPIs.
